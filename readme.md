## Installation

Clone repositori:
```bash
  git clone https://github.com/zidaaja/project-msib-ecommerce.git
```

Pindah ke direktori project:
```bash
  cd project-msib-ecommerce
```

Install dependensi:
```bash
  npm install
```

Jalankan project:
```bash
  npm run start
```
